# 星港アトラス HOME キービジュアル

## ゲームフック

`星港アトラス` は、空に浮かぶ港町を拠点に、船団を編成しながら探索と交易を進めるスカイポート運営 RPG。

## HOME 画面の意図

HOME 画面では以下を一目で伝える。

- 空港都市を拠点に日課と出撃を回すゲームであること
- 情報量は多いが、主導線は迷わず押せること
- `出航する` が最優先 CTA であること
- 背景だけでなく、主要 UI を載せた完成画面モックとして世界観と情報密度を示すこと

## 採用した KV 方針

- 背景アート単体ではなく、`上部ステータス` `プロフィール` `イベント` `任務` `主導線ボタン` `下部ナビ` まで含めた画面KVにする
- 実運用の HOME 画面を想定し、中央下に `出航する` を強く置く
- 右上参照で見ても構造が伝わるよう、遠景の街並みと手前のUIを明確に分離する
- 雰囲気は `蒼空スチームファンタジー` だが、宣伝ポスターではなく日常的に見るゲーム内ホームとして整理する
