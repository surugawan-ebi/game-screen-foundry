# 星港アトラス HOME 画面サンプル

試作用の複雑な 1 画面サンプル。  
架空ゲーム `星港アトラス` の HOME 画面を想定している。
キービジュアルは背景画ではなく、主要 UI を載せた完成画面モックとして構成している。

画面構成:

- 背景
- 上部ステータスバー
- プレイヤープロフィールカード
- デイリーミッション枠
- イベントバナー
- メイン導線ボタン
- ギフト / ガチャ導線
- 下部ナビゲーション

今回のサンプルは、完成KVをそのまま1枚で再現するのではなく、`本番実装で差し替えや再利用が効く粒度` まで分解している。
特に `上部HUB` `左プロフィール` `右デイリー任務` は、中身パーツまで placement に出している。

このフォルダは、そのままベータ版の `Load From Folder` で読み込める。

同梱ファイル:

- `key-visual.png`: この画面のキービジュアル
- `key-visual.md`: 画面意図
- `assembly-design.md`: なぜその粒度で分解したか
- `screen-kv.json`
- `material-spec.json`
- `world-preset.json`
- `bundle.json`
