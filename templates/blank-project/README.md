# Blank Project Template

This template is a minimal external creative project that can be loaded by Game
Screen Foundry.

Load it from the browser input with:

```text
/absolute/path/to/game-screen-foundry/templates/blank-project/creative
```

Or load the screen explicitly:

```text
/absolute/path/to/game-screen-foundry/templates/blank-project/creative#home
```

Copy the `creative/` folder into a game repository, then replace the sample
screen JSON and generated PNGs with project-specific data.
